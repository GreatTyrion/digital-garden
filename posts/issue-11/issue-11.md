## 知识分享：React 满分核弹级漏洞 (React2Shell) 技术分析

**简介**：这是一个被评为 CVSS 10.0（最高危级别）的严重安全漏洞，影响 React 19.x 和 Next.js 15.x/16.x 版本。

### 核心要点

- **漏洞编号**：CVE-2025-55182 (React) / CVE-2025-66478 (Next.js)
- **漏洞类型**：CWE-502（不可信数据的反序列化）
- **影响范围**：所有使用 React Server Components (RSC) 的框架
- **攻击后果**：攻击者可在无需身份验证的情况下，通过单个 HTTP POST 请求实现远程代码执行 (RCE)

### 技术背景

该漏洞源于 React Server Components 的 Flight 协议在反序列化过程中的安全缺陷。攻击者通过原型链污染技术，可以获取 JavaScript 的 Function 构造器，从而在服务器端执行任意代码。

### 修复建议

React 团队已发布紧急补丁，建议所有受影响的项目立即升级到安全版本。如果无法立即更新，应采取临时缓解措施，如禁用或限制 RSC 端点访问。

**详细分析**：查看完整的[技术分析报告](./react-ten-point-bug.md)，深入了解漏洞原理、攻击链条和防护措施。

## 工具分享：谷歌 NotebookLM - 将信息变为创意的个人 AI 助理

**简介**：NotebookLM 是谷歌推出的个人 AI 研究助理，基于 RAG（检索增强生成）技术，严格基于你提供的资料回答问题，杜绝 AI 幻觉。

### 核心特性

- **高可靠性**：基于你上传的资料回答问题，每个答案都会标注来源
- **多格式支持**：轻松处理 PDF、Google 文档、YouTube 视频链接、音频文件等多种信息来源
- **强大的内容生成能力**：不仅仅是整理笔记，更能一键生成简报、视频、播客等多种内容

### 主要功能

#### 信息管理

- **自动总结**：上传资料后立即生成核心要点总结
- **精准问答**：像与真人对话一样，向你的资料提问
- **深度研究**：AI 驱动的深度研究功能，自动寻找补充来源并生成研究报告
- **知识可视化**：将笔记转化为思维导图和信息图表

#### 内容创作

- **演示文稿生成**：一键生成专业级 PPT，支持多种视觉风格定制
- **视频与播客**：将文字资料转化为生动的音视频内容
- **学习辅助**：自动生成学习卡片和测验，帮助学生巩固知识点

**详细指南**：查看完整的[使用指南](./a-personal-AI-assistant-that-turns-information-into-creativity.md)，深入了解如何利用 NotebookLM 从信息管理大师晋升为内容创作高手。
